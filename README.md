# 🎄 Advent of Code 2024

https://adventofcode.com/

📁 Create a daily folder

```zsh
pnpm new 01 # 01 - 25
```

🚨 Run ESLint

```zsh
pnpm lint # lint & fix
```

🎨 Run Prettier

```zsh
pnpm prettier # fix
```

🧪 Run Vitest

```zsh
pnpm test # test once
pnpm watch # watch mode
```

🐕 Run Husky

```zsh
pnpm husky # init husky
```
